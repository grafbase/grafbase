## Fix

- Introspection schema: added `includeDeprecated` input field to `__Type.inputFields`, `__Field.args`  and `__Directive.args`.
- Introsepction: implement `includeDeprecated` filter for `__Type.inputFields`
- Introsepction: implement `includeDeprecated` filter for `__Field.args`

> Note: `__Directive.args` field now allow to pass `includeDeprecated` filter, but directives support is not fully implemented yet, so an introspection query that queries `__schema.directives` will return `[]`. (see https://github.com/grafbase/grafbase/issues/3596)
