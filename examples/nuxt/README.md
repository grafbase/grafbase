## Getting Started

1. Run `npx degit grafbase/grafbase/examples/nuxt grafbase-with-nuxt` to clone this example
2. Change directory into the new folder `cd grafbase-with-nuxt`
3. Run `cp .env.example .env`
4. Open `.env` in your code editor, and provide your Grafbase API endpoint and API key
5. Run `npm install`, or `yarn install` to install dependencies
6. Run `npm run dev`, or `yarn dev` and visit http://localhost:3000
